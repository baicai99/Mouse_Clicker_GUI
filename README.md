1. 支持 连击
2. 支持 双击
3. 支持 范围内 双击、连击
1. 支持本地化

    - 印度: हिन्दी (hi)  
    - 印度尼西亚: Indonesian (id)  
    - 巴基斯坦: Urdu (ur)  
    - 葡萄牙: Portuguese (pt)  
    - 西班牙: Español (es)  
    - 俄罗斯: Русский (ru)  
    - 孟加拉国: বাংলা (bn)  
    - 阿拉伯国家: العربية (ar)
